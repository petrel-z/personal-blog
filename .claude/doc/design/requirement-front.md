# 个人博客系统 — 前台需求文档（PRD）

版本：v1.1
模块：前台系统（Visitor Side）
负责人：博主
更新日期：2026-03-18

---

# 一、项目背景

构建一个面向技术读者的个人博客前台系统，用于：

- 展示技术文章

- 提供良好的阅读体验

- 支持互动（评论 / 点赞）

- 提供内容发现能力（搜索 / 排行榜 / 分类）

- 建立个人技术品牌


---

# 二、项目目标

## 2.1 核心目标

- 提升阅读体验

- 提高文章可发现性

- 提供基础互动能力

- 支持移动端访问

- 支持暗黑模式

- 支持国际化扩展


## 2.2 成功指标（可选）

- 平均阅读时长提升

- 评论转化率提升

- 搜索使用率

- 页面加载时间 < 2s


---

# 三、用户角色

## 3.1 访客

特征：

- it 技术从业者或学习者

- 通过搜索引擎或社交分享进入

- 主要行为：阅读 / 搜索 / 评论 / 点赞 / 分享


---

# 五、功能模块说明

---

# 5.1 首页（文章列表页）

## 功能目标

展示最新文章，提供内容入口。

## 页面内容

每篇文章卡片包含：

- 标题

- 摘要（优先使用文章概述，否则截取前 200 字）

- 分类

- 标签

- 发布时间

- 阅读量

- 点赞数

- 评论数

- 封面图（可选）

- 预计阅读时间（字数 / 300）


## 交互行为

- 点击标题进入详情页

- 支持分页（每页 10 篇）

- 默认按发布时间倒序


## 性能要求

- 首屏加载 < 2 秒

- 图片懒加载

- 支持缓存


优先级：高

---

# 5.2 文章详情页

## 页面结构

```
标题
文章元信息
正文
目录（右侧）
点赞区
评论区
相关推荐
上一篇/下一篇导航
```

---

## 正文渲染要求

- 支持 Markdown 渲染

- 支持代码高亮

- 代码块一键复制

- 图片自适应

- 自动生成目录（基于 h1/h2/h3）


目录要求：

- 可折叠

- 平滑滚动定位

- 当前章节高亮

- 移动端隐藏为抽屉


---

## 阅读量统计

规则：

- 同一 IP 10 分钟内只计一次

- 前端展示累计值

- 进入页面时自动触发计数


---

## 点赞功能

- 每 IP 每篇仅一次

- 实时更新点赞数

- 防刷机制（IP + 时间窗口限制）

- 点赞后按钮禁用，提示"已点赞"


---

## 评论功能

### 输入规则

- 昵称必填（1-20 字符）

- 头像可选（默认 Gravatar 头像）

- 内容 ≤ 500 字

- 图片验证码校验（4 位字符，大小写字母 + 数字）

- 频率限制：1 分钟 1 条/IP

### 展示规则

- 按时间倒序

- 支持一级回复（嵌套展示）

- 支持分页（每页 10 条）

- 显示昵称、头像、时间、内容


### 安全机制

- XSS 过滤（前端转义 + 后端过滤）

- 敏感词过滤（自动标记待审核）

- 频率限制（1 分钟 1 条/IP）

- 验证码防 spam


### 验证码规格

- 类型：图片验证码

- 格式：4 位字符（大小写字母 + 数字组合）

- 功能：支持点击刷新、支持键盘空格键刷新

- 有效期：5 分钟

- 错误处理：错误 3 次后强制刷新验证码


优先级：高

---

## 上一篇 / 下一篇导航

### 功能目标

提升阅读连续性，增强同主题文章浏览体验。

---

### 展示位置

位于文章底部，评论区上方。

示例：

```
← 上一篇：xxx 标题
        下一篇：xxx 标题 →
```

---

### 逻辑规则

#### 情况一：从分类页进入文章

- 上一篇 / 下一篇在"当前分类"内切换

- 按发布时间排序（与分类页一致）

- 若为当前分类第一篇：上一篇按钮隐藏或置灰

- 若为当前分类最后一篇：下一篇按钮隐藏或置灰


---

#### 情况二：从首页或搜索页进入文章

- 上一篇 / 下一篇按全站时间顺序切换

- 默认按发布时间倒序

---

### 交互行为

- 点击按钮无整页刷新（可选，根据技术实现决定）

- 自动滚动到文章顶部

- URL 更新为新文章地址


---

### 边界处理

- 若无上一篇或下一篇，对应按钮置灰

- 避免跨分类跳转（仅在分类上下文内切换）

---

# 5.3 分类页

## 5.3.1 分类总览页

路径：

```
/categories
```

### 功能说明

展示所有文章分类，供访客按主题浏览。

### 页面展示内容

每个分类卡片包含：

- 分类名称

- 分类描述（可选）

- 当前分类文章数量

- 最近更新时间（可选）


### 交互行为

- 点击分类 → 进入该分类的文章列表页


优先级：中

---

## 5.3.2 分类文章列表页

路径：

```
/category/{slug}
```

### 功能说明

展示当前分类下的文章列表。

### 页面展示内容

与首页文章列表一致：

- 标题

- 摘要

- 发布时间

- 阅读量

- 点赞数

- 评论数

- 预计阅读时间

- 封面图（可选）


### 排序规则

- 默认按发布时间倒序

- 支持分页（每页 10 篇）


### 交互行为

- 点击文章 → 进入文章详情页

- 保留当前分类上下文（用于上一篇/下一篇切换）

优先级：高

---

# 5.4 搜索页

## 功能

- 支持标题 + 内容全文搜索

- 支持模糊匹配

- 高亮关键词（搜索词高亮）

- 无结果提示（"未找到相关文章，试试其他关键词"）


## 交互

- 输入防抖（500ms）

- 回车触发

- 支持清空

- 支持搜索历史记录（本地存储，可选）


优先级：高

---

# 5.5 热度排行榜页

## 功能

展示热门文章前 10。

## 排序规则

热度计算公式：

```
热度 = 阅读量 × 0.6 + 点赞 × 0.3 + 评论 × 0.1
```

## 榜单类型

- 总榜

- 本月榜


优先级：中

---

# 5.6 创作记录页

灵感来源：GitHub 提交日历

## 功能

- 按天统计发布 / 编辑次数

- 热力图展示

- 支持年份切换

- 点击某日查看详情（弹出当天操作列表）


## 时间轴

展示：

- 发布文章

- 编辑文章

- 删除文章


优先级：中

---

# 5.7 关于我页面

功能：

- 展示个人介绍

- 支持 Markdown

- 展示头像

- 社交链接（GitHub、Twitter、微博等）


优先级：低

---

# 5.8 主题切换

支持：

- 浅色模式

- 深色模式

- 自动跟随系统


规则：

- 用户选择持久化（localStorage）

- 页面刷新保持一致


优先级：中

---

# 5.9 多语言支持

支持：

- 中文

- 英文


范围：

- 菜单

- 系统提示

- 按钮文案


不包含：

- 文章自动翻译


优先级：低

---

# 5.10 分享功能

支持：

- 复制链接

- 微信分享（生成二维码）

- 微博分享（跳转分享页）


分享内容包含：

- 标题

- 摘要

- 封面图


优先级：低

---

# 5.11 RSS 支持

路径：

```
/rss.xml
```

规则：

- 发布文章自动更新

- 包含标题 / 链接 / 摘要 / 时间

- 支持浏览器识别


优先级：低

# 六、非功能性需求

---

## 6.1 性能要求

- 首页加载 < 2 秒

- Lighthouse 性能评分 > 90

- 图片压缩（质量 80%）

- 图片懒加载（滚动到可视区域加载）

- 代码分割（按路由拆分）

- 接口响应时间 < 500ms


---

## 6.2 安全要求

- 防 XSS（输入转义 + 输出过滤）

- 防 CSRF（Token 验证）

- 评论限流（1 分钟 1 条/IP）

- 点赞防刷（IP + 时间窗口）

- 阅读量防刷（IP + 10 分钟窗口）

- 敏感词过滤（评论自动审核）


---

## 6.3 兼容性

- 支持主流浏览器（Chrome、Firefox、Safari、Edge 最新版本）

- 移动端优先设计

- 响应式布局（断点：768px、1024px、1280px）


---

# 七、异常处理

## 7.1 页面异常

- 404 页面：访问路径不存在时展示自定义 404 页面

- 服务器异常：展示友好的错误提示和重试按钮


## 7.2 内容异常

- 搜索无结果：展示"未找到相关文章"提示

- 文章加载失败：展示错误提示和重试按钮

- 图片加载失败：展示默认占位图


## 7.3 网络异常

- 网络断开：展示网络错误提示

- 请求超时：展示超时提示和重试选项


---

# 八、错误提示文案规范

## 8.1 评论相关

| 场景 | 错误提示 |
|------|----------|
| 昵称为空 | 请输入昵称 |
| 昵称过长 | 昵称长度不能超过 20 字符 |
| 评论内容为空 | 请输入评论内容 |
| 评论内容过长 | 评论内容不能超过 500 字 |
| 验证码错误 | 验证码错误，请重新输入 |
| 验证码已过期 | 验证码已过期，请点击刷新 |
| 评论频率过高 | 评论太频繁，请稍后再试 |
| 评论提交失败 | 评论提交失败，请检查网络后重试 |
| 评论等待审核 | 评论提交成功，等待审核后将显示 |

## 8.2 通用错误

| 场景 | 错误提示 |
|------|----------|
| 网络错误 | 网络连接失败，请检查网络设置 |
| 服务器错误 | 服务器开小差了，请稍后再试 |
| 数据加载失败 | 数据加载失败，请刷新页面重试 |
| 操作失败 | 操作失败，请稍后重试 |

## 8.3 成功提示

| 场景 | 成功提示 |
|------|----------|
| 评论提交成功 | 评论提交成功，等待审核后将显示 |
| 点赞成功 | 已点赞 |
| 复制成功 | 链接已复制 |
